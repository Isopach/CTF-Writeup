# repeaaaaaat

Landing page is a single image repeated dozens of times, increasing as you scroll.

Let's take a look at the source code, which has the base64 encoded comment that points to `/?secret=flag` when decoded.

We try appending it to the URL, and discover that the string `flag` is appended at the bottom of the page, after the div.     
Various strings work and they all get reflected.         

Maybe there is an SSTI, so let's try `{{7*7}}`.

It returns 49! Template Injection confirmed.
